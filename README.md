# Школьная столовая - Система управления питанием

Веб-приложение для учёта и контроля питания в школьной столовой с поддержкой различных ролей пользователей.

## Краткое описание проекта

Это веб-приложение разработано для автоматизации процессов в школьной столовой. Система позволяет учащимся просматривать меню, оплачивать питание и оставлять отзывы, поварам отслеживать выдачи и управлять запасами, а администраторам контролировать всю деятельность столовой. Приложение обеспечивает удобный интерфейс для всех ролей пользователей и надежную систему управления данными.

## Инструкция по установке/развертыванию

### Требования
- Docker и Docker Compose
- Git
- Node.js 18+ (для локальной разработки)

### Развёртывание с Docker

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Name341/cafeteria-management
cd cafeteria-management
```

2. Скопируйте файл переменных окружения:
```bash
cp .env.example .env
```

3. Запустите приложение:
```bash
docker-compose up -d --build
```

4. Приложение будет доступно:
   
   **На этом компьютере:**
   - Frontend: http://localhost:3000
   - API: http://localhost:5000
   
  
### Локальная разработка

#### Backend

```bash
cd backend
npm install
npm start
```

#### Frontend

```bash
cd frontend
npm install
npm start
```
